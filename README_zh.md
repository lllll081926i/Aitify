# AI CLI Complete Notify

一个轻量级的 AI CLI 任务完成通知工具

## 功能特性

- 🖥️ Windows 原生桌面通知
- 🎯 智能监控 Claude / Codex / Gemini 日志
- ⏱️ 可配置的任务时长阈值
- 🚀 开机自启动
- 🌐 中英文界面

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 系统要求

- Windows 10/11
- Node.js 18+
- Rust 1.77+

## 工作原理

监控 AI CLI 日志文件,检测任务完成信号,发送 Windows 原生通知。
